# server-Node.js-demo

这是一个使用 Node.js 创建的服务器，提供 HTTP 服务。

### 使用说明

在 Git Bash 命令行中输入 `node server.js 端口号` ，即可开启服务器，并监听当前的端口号。之后在浏览器中输入 `http://localhost:端口号/[ path ]` ,即可看到相应的信息。

本服务器提供3个 `path` ,分别是 `/index`、`/style`、`/javascript` 

`/index` 响应内容为 `HTML`

`/style` 响应内容为 `css`

`/javascript` 响应内容为 `javascript`
